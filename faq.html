📦 Package

Course content তিনটিতেই হুবহু same — ৯৯+ class, ৫টা module।

  • Basic ($39): শুধু recorded classes — নিজের মতো শিখুন
  • Pro ($99): Basic + 1 month live support + $5K funded account সহায়তা
  • Elite ($199): Pro-এর সব + 3 months live support + $10K funded account + Priority mentoring

নিজে নিজে পারবেন মনে করলে Basic যথেষ্ট। হাতে ধরে গাইড চাইলে Pro বা Elite।

হ্যাঁ, যেকোনো সময়। শুধু difference amount pay করলেই হবে:

  • Basic → Pro: আরও $60 (= $99 − $39)
  • Basic → Elite: আরও $160 (= $199 − $39)
  • Pro → Elite: আরও $100 (= $199 − $99)

Upgrade করতে চাইলে Telegram-এ বলুন।

সহজ rule:

  • Budget কম / নিজে শিখতে comfortable: Basic ($39)
  • Live guidance চান + Funded account লক্ষ্য: Pro ($99)
  • Maximum support + Serious trader হতে চান: Elite ($199)

সিদ্ধান্ত নিতে না পারলে Telegram-এ আপনার situation বলুন — recommend করে দেওয়া হবে।

📚 Course

হ্যাঁ, একদম। Module 01 — Basic Foundation একদম zero থেকে শুরু হয়। Trading কী, Chart কীভাবে পড়তে হয়, Broker কীভাবে কাজ করে — সব step by step।

কোনো আগের knowledge লাগবে না। শুধু শেখার ইচ্ছা থাকলেই যথেষ্ট।

৫টি module-এ পুরো curriculum:

  • Module 01: Basic Foundation — Chart, Broker, Orders, Crypto Wallet
  • Module 02: ICT Concept — Liquidity, Market Structure, Premium/Discount, Strategies
  • Module 03: Fundamental Analysis — Central Banks, Macro, News Trading, COT
  • Module 04: Psychology — Mindset, FOMO/Greed/Fear, Journaling
  • Module 05: Prop Trading — Firm Selection, Challenge Rules, Funded Account

Lifetime access আছে — তাড়া নেই।
